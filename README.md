# Hackathon_data_wrangling_ext

## Project Overview
This repository contains material for the AMMnet Hackathon on Data wrangling extension focused on **data wrangling, cleaning, and preparation**. The goal is to transform raw data into a clean, analysis-ready dataset and document the steps clearly so the work is reproducible and easy to extend.

## Objectives
- Import raw data from one or more sources.
- Clean and standardize variables, formats, and missing values.
- Reshape and merge datasets where needed.
- Create a reproducible workflow for analysis.

## Tools and Technologies
- R / RStudio
- Quarto / Markdown
- Git / GitHub
- Tidyverse, or other data-cleaning libraries

## Workflow
1. Load the raw data.
2. Inspect structure, data types, and missingness.
3. Clean column names and values.
4. Handle duplicates and inconsistencies.
5. Create final cleaned datasets.
